# AI Customer Service

React Native (Expo) app for the AI customer service project.

## Setup & run

The app lives in the `app` folder (Expo requires a URL-friendly project name).

```bash
cd app
npm start
```

Then:

- **iOS simulator:** press `i` in the terminal or run `npm run ios`
- **Android emulator:** press `a` or run `npm run android`
- **Web:** press `w` or run `npm run web`

## Requirements

- Node.js 18+
- For iOS: Xcode and iOS Simulator (macOS)
- For Android: Android Studio and an emulator or device
- [Expo Go](https://expo.dev/go) on a physical device (optional)

## Project structure

- `app/` – Expo React Native app (Expo Router, React Navigation)
- `app/app/` – File-based routes and screens
